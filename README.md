# Wind
### About:
This is a game about growing trees by shooting seeds into the ground.

### Languages:
- C++

### Frameworks:
- SFML
- Box2D

### Current Features:
- Random terrain generation.
- Wind with a visible component.
- Shooting seeds from the seed cannon.
- Variable shot power visible in UI.
